这个是我的毕业设计，还有许多需要改进的地方。前端和安卓端是我大二时写的课程设计修改了一下拿过来用的。

采用微服务架构拆分检测复核服务与违规申诉服务，使用 Nacos 实现服务注册与发现，结合 OpenFeign 完成服务间远程调用，并引入 Spring Cloud Gateway 作为统一网关，实现路由转发、跨域处理及接口统一入口。

基于 SmartJavaAI 框架集成 YOLOv8 模型，将摄像头端（安卓）上传的关键帧封装为检测任务并投递至 RabbitMQ 消息队列，通过消息监听器异步消费任务完成复核检测；对检测结果进行目标关联，并结合业务规则完成违规判定与结果落库。

使用 Token + Redis + RBAC 实现登录认证、会话管理与权限控制，前端基于 Vue 完成违规信息展示、查询与申诉处理页面。
